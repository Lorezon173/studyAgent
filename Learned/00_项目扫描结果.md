## 2. 项目类型判断

**单体应用（多入口、模块化非常成熟）**

判断依据：
- 单一仓库、单一 `pyproject.toml`，没有多服务子目录
- 但有 **5 个独立入口**（见下文），不止 Web 一种
- `slo/` 是独立 CLI 模块（`uv run python -m slo.run_regression`），更接近"工具"而非业务服务

由于规模与文档完整度，本项目实质上是一个 **"具备 SLO 门禁、运维 runbook 的可运营学习 Agent 系统"**（README.md:115、254）。

## 8. 项目核心定位（来自 README.md:8-18）

LearningAgent 以**费曼学习法**为主线：

```
诊断已有认知 → 易理解地讲解 → 要求复述并评估 → 针对错误追问与补救 → 输出总结与复习建议
```

**它不是单轮聊天助手，而是"可沉淀学习轨迹、具备检索能力、带有运维基线"的学习编排系统。**
